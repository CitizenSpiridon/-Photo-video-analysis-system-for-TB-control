Автоматизированная система контроля применения СИЗ на строительных площадках на базе YOLOv11

Репозиторий содержит программную реализацию практической части выпускной квалификационной работы (ВКР), посвященной разработке интеллектуальной системы видеоаналитики для фиксации нарушений требований охраны труда (детекция строительных касок и сигнальных жилетов).

В основе комплекса лежит кастомная модель компьютерного зрения YOLOv11, дообученная на специализированном датасете Construction-PPE. Система работает в режиме реального времени, обрабатывая видеопоток с камер, и автоматически отправляет уведомления о нарушениях в Telegram.

🚀 Ключевые особенности системы

Современная архитектура: Использование легковесных и быстрых моделей YOLOv11-nano и YOLOv11-small, оптимизированных для работы без мощных серверных GPU.

Умное логирование инцидентов: Система не перегружает память постоянной видеозаписью. Кадр сохраняется на диск и отправляется инженеру по охране труда только в момент фиксации нарушения.

Двухуровневая проверка (Многоклассовая детекция): Модель распознает классы: person (человек), helmet (каска), vest (жилет), а также их отсутствие (no_helmet, no_vest).

Мгновенные алерты: Автоматическая отправка тревожных кадров с рамками детекции (bounding boxes) в Telegram-канал или чат специалиста через Telegram Bot API.

📋 Требования к системе

Python 3.9 — 3.11

Веб-камера, IP-камера (RTSP-поток) или тестовый видеофайл

Токен Telegram-бота (полученный у @BotFather)

💻 Инструкция по установке и запуску

Выберите инструкцию ниже в зависимости от вашей операционной системы.

🍏 Инструкция для macOS

1. Откройте Терминал (нажмите Cmd + Пробел, введите Терминал и нажмите Enter).

2.Перейдите в папку с проектом:
bash cd "/Users/sashka/Desktop/TB_bot_nano 2"

3. Создайте виртуальное окружение venv:
bash python3 -m venv .venv

4. Активируйте виртуальное окружение:
bash source .venv/bin/activate
(После активации в начале строки терминала появится маркер (.venv))

5. Обновите менеджер пакетов pip:
bash python -m pip install --upgrade pip

6. Установите необходимые зависимости:
bash pip install ultralytics pyTelegramBotAPI opencv-python

7. Запустите систему:
bash python main.py
(Замените main.py на имя вашего главного файла, если оно отличается)

🪟 Инструкция для Windows

1. Откройте Командную строку: нажмите клавиши Win + R на клавиатуре, введите cmd в появившемся окне и нажмите Enter.

2. Перейдите в папку с вашим проектом:
cmd cd "C:\Users\Имя_Пользователя\Desktop\TB_bot_nano 2"
(Замените Имя_Пользователя на имя вашей учетной записи в Windows)

3. Создайте виртуальное окружение venv:
cmd python -m venv .venv

4. Активируйте виртуальное окружение:

Если вы работаете в обычной Командной строке (cmd):
cmd .venv\Scripts\activate.bat

Если вы используете PowerShell:
powershell .venv\Scripts\Activate.ps1
(После успешной активации слева от командной строки появится надпись (.venv))

5. Обновите менеджер пакетов pip:
cmd python -m pip install --upgrade pip

6. Установите все необходимые библиотеки:
cmd pip install ultralytics pyTelegramBotAPI opencv-python

7. Запустите программу:
cmd python main.py

⚡️ Быстрый запуск в один клик (только для Windows)

Чтобы не вводить команды в консоль каждый раз при запуске проекта:

Создайте в папке проекта обычный текстовый файл и назовите его run.bat.

Откройте его через Блокнот и вставьте следующую строчку:
cmd @echo off && call .venv\Scripts\activate && python main.py && pause

Сохраните файл. Теперь для запуска проекта вам достаточно просто дважды кликнуть по файлу run.bat.

⚙️ Настройка конфигурации

Перед запуском убедитесь, что в коде (или файле конфигурации .env) указаны ваши корректные данные:

TOKEN — токен вашего Telegram-бота.

CHAT_ID — ID чата или канала, куда бот будет присылать уведомления.

model_path — путь к файлу весов вашей обученной модели (например, best.pt).
